# 🍕 Pizza Sales Website

Pizza Sales Website is a modern restaurant website concept designed to promote pizza products, showcase menu items, and encourage online ordering.

The website focuses on a clean layout, engaging visuals, and simple navigation for users who want to quickly explore the menu and order food.

## 🚀 Live Demo
https://v0-pizza-sales-website-silk.vercel.app/

## 📌 Features
- Pizza menu and product showcase
- Promotions and special offers section
- Modern and responsive website layout
- Clear call-to-action for ordering
- Clean and simple UI

## 🛠️ Tech Stack
- TypeScript
- CSS
- JavaScript

## 🎯 Project Purpose
This project was built as a portfolio example of a modern food delivery or restaurant website to demonstrate front-end development and UI layout skills.
